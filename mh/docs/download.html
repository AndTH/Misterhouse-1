Where to Download the latest version of MisterHouse

Updated to version 2.34 on 11/12/00:

  • If using a Windows:
    • misterhouse_src_234.zip : The Perl source files and documentation. About 2 Meg. If you have Perl installed, this is all you need.
    • misterhouse_win_234.zip : The Windows compiled file. About 2 Meg. This will run stand alone without Perl installed on Windows 95,98, NT, or 2K. Note: the above misterhouse_src_234.zip is also required, as it has documentation and support files:
  • If using a Unix, use either of the following, each about 2 Meg:
    • misterhouse-2.34.tar.gz
    • misterhouse-2.34-1.noarch.rpm

You can get older versions of MisterHouse on the Soureforge download page

All the code is also available at misterhouse.net/mh and there is a MisterHouse CVS repository (used by co-developers) complements of our friends at SourceForge.
